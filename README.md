# shuai blog

##写文章

```bash
hexo new [layout] <title>
```

layout:

| layout | path |
| ----- | --------------- |
| post  | source/\_posts  |
| page  | source          |
| draft | source/\_drafts |

##编译

```bash
hexo generate
```

##部署(直接部署到远程 web 托管)

```bash
hexo deploy
```

##编译+部署

```bash
hexo g -d
```
